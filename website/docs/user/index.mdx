# How to use the xPack Build Box

This section is intended for those who plan to use this
module in their
own projects.


The `xpack-build-box` module can be imported into both TypeScript
and JavaScript Node.js code.

In TypeScript and ECMAScript modules, use `import`:

```typescript
import { Xxx } from 'xpack-build-box'
```

In JavaScript with CommonJS, use `require()`:

```javascript
const { Xxx } = request('xpack-build-box')
```
